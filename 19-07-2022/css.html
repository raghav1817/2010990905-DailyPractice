Extensions are normally not allowed to pick colors from local file:// URL addresses.
If you really need it, you can enable Allow access to file URLs option in Chrome Extensions page.
Because this allows extension to access your local files and thus introducing potentional security problem,
this can't be done from extension settings, but must be explicitly allowed through Chrome's settings.
